# 3D 渲染圖轉換成 CAD 圖 — 浴缸互動設計原型

## 使用方式
直接雙擊 `index.html` 用瀏覽器開啟（需網路載入 3D 引擎）。

## 流程概念
1. **客戶端**：網頁上選造型（圓角矩形／圓端形／橢圓形）、調整尺寸、壁厚、排水位置、顏色，3D 即時預覽並自動計算容量與重量。
   - **不對稱設計**（對應 38.JPG 照片款）：靠背增高（後緣高、缸緣斜面）、蛋形係數（前窄後寬）、底部收縮（側壁內收）。點「★ 套用照片款」一鍵重現。
   - 3D 視圖可 360° 自由拖曳旋轉（含仰視）、滾輪縮放。藍色半透明形狀＝水位模擬，可勾選開關。
   - 網址參數 `?preset=photo` 可直接分享照片款設定。
   - **✏️ 手繪模式**：客戶可直接在網頁畫布上畫缸口形狀（或上傳紙上手稿照片，自動輪廓辨識），加選手繪側面牆壁剖面曲線，系統自動平滑化並建成 3D，做出獨一無二的浴缸。手繪資料會一併輸出到 DXF 與 JSON 規格表。
2. **輸出**：
   - `下載 CAD 圖檔 (DXF)` → 含俯視／前視／側視三視圖與標註，AutoCAD 可直接開啟修改。
   - `下載規格表 (JSON)` → 完整參數，供設計師或系統讀取。
3. **設計端 → 製造端**：設計師依 DXF 微調後進行模具開發。

## 後續可擴充
- 嵌入式浴缸類型與安裝尺寸
- STL 三維模型輸出（供模具 3D 軟體）
- 直接嵌入公司網站（單一 HTML 檔即可嵌入 iframe）
- 客戶送出訂單後自動 Email 給設計師
